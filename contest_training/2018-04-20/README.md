# 题解
## A 直角三棱锥
求一个直角三棱锥中的所有点（包括边上的点）。
很简单的题。注意取模，否则会爆longlong。
## B 前缀查询
很明显的一道字典树的题。注意在字典树上加 lazy 标记。
## C 可达性
题意大致是有一个有向有环图（无重边与子环）求最小的点集能覆盖掉所有的点。
做法就是 tarjan 缩点后找出所有入度为 0 的点输出。注意缩点时注意点值为强连通中最小的值。
