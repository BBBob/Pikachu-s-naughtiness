# CDQ分治

## BZOJ 2683 简单题
题目链接：这是权限题……
### 题意
对于矩阵，有两种操作：
1、查询$ (x1,y1) $到$ (x2,y2) $的矩形的面积。
2、使$ (x1,y1) $的值加V。
对于每个矩阵有多个操作。
### 题解
将查询分解为对与四个以$ (1, 1) $为起点的矩形数值和。
对于每个查询$ (x,y) $只有修改操作$ (x1, y1), x1<=x && y1<=y $且修改在查询前才会有贡献。
所以对于$ x $维度排序
所以对于时间维度分治
所以对于$ y $维度用树状数组统计
每次：
1、递归左区间
2、计算当前区间左区间对有区间的贡献
3、递归右区间
